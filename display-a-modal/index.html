This page demonstrates how to progressively enhance going from an index page to a detail page. Specifically, instead of leading the user to another page, clicking the image link below will open a modal with the enlargement inside. Clicking outside the enlargement or on the X closes the enlargement.

Expressed in terms of acceptance criteria:

  • Given the photograph is visible as a thumbnail on the index page,
    When the user clicks on the thumbnail
    Then the photograph is enlarged in a modal
  • Given the photograph is enlarged in a modal,
    When the user clicks outside the modal
    Then the modal is closed
  • Given the photograph is enlarged in a modal,
    When the user clicks on the X
    Then the modal is closed
